# Aduana 4.0 - Sistema de Control Fronterizo - Ev Transversal ing de software

Aduana 4.0 es un prototipo funcional de alta fidelidad diseñado para digitalizar y automatizar los trámites de cruce fronterizo en el Paso Los Libertadores (Chile–Argentina). El sistema centraliza la gestión de registro vehicular, control fitosanitario (SAG), control migratorio (PDI) y la autorización de viaje para menores de edad en una sola plataforma interactiva con enrutamiento dinámico según el rol del usuario.

Normas de Referencia: IEEE 829-2008, ISO/IEC 29119, ISO/IEC 25010.
