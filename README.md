# Aduana 4.0 - Sistema de Control Fronterizo
Prototipo funcional de alta fidelidad para digitalizar
los procesos del Paso Fronterizo Los Libertadores.
## Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS v4
- lucide-react
README.md
- sessionStorage
## Instalación
git clone https://github.com/rzuniga/aduana-4.0.git
cd aduana-4.0
npm install
npm run dev
## Roles disponibles
- Ciudadano
- Funcionario de Aduana
- Funcionario SAG/PDI
- Administrador
## Características
- Validación RUT (módulo 11)
- Validación patentes (BBBB99 / AA9999)
- Persistencia resiliente (sessionStorage)
- Validación cruzada menores
- 4 dashboards diferenciados por rol
